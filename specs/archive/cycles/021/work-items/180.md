# WI-180: Add checkSchemaVersion version-mismatch and version-current tests

## Context

This work item addresses Q-70 from cycle 020 review.

`checkSchemaVersion` in `mcp/artifact-server/src/schema.ts` has 3 branches:

1. **Version-0 (fresh DB)**: `user_version === 0` → returns `true`. **Tested** (added by WI-177).
2. **Version-mismatch (stale DB)**: `user_version !== 0 && user_version !== CURRENT_SCHEMA_VERSION` → closes DB, deletes DB file + WAL + SHM files, returns `false`. **Untested.**
3. **Version-current**: `user_version === CURRENT_SCHEMA_VERSION` (6) → returns `true`. **Untested.** This is the path executed on every server startup after a successful migration.

## File to Modify

`mcp/artifact-server/src/__tests__/schema.test.ts`

The file imports `{ createSchema, checkSchemaVersion }` from `"../schema.js"`. It uses `new Database(":memory:")` for in-memory databases and imports `{ describe, it, expect }` from `"vitest"`.

### Test for version-mismatch branch

This test requires a real on-disk file because the function deletes the file. Use a temp path.

```typescript
import { existsSync, mkdtempSync, rmSync } from "fs";
import { tmpdir } from "os";
import { join } from "path";
```

Add these imports at the top of the file if not already present. (`fs` and `os` are Node built-ins, no new dependencies needed.)

Test body:
```typescript
it("returns false and deletes the database file when user_version is stale", () => {
  // Create a temp directory and a real SQLite file within it
  const dir = mkdtempSync(join(tmpdir(), "ideate-schema-test-"));
  const dbPath = join(dir, "test.db");

  // Write a stale version into a real SQLite file
  {
    const db = new Database(dbPath);
    db.pragma("user_version = 5"); // stale — current is 6
    db.close();
  }

  // Reopen and check schema version
  const db = new Database(dbPath);
  const result = checkSchemaVersion(db, dbPath);

  // Function should return false and delete the file
  expect(result).toBe(false);
  expect(existsSync(dbPath)).toBe(false);

  // Cleanup temp dir (file is already gone)
  rmSync(dir, { recursive: true, force: true });
});
```

### Test for version-current branch

Use in-memory DB with `user_version` set to `CURRENT_SCHEMA_VERSION` (6).

```typescript
it("returns true when user_version matches CURRENT_SCHEMA_VERSION", () => {
  const db = new Database(":memory:");
  db.pragma(`user_version = 6`); // matches CURRENT_SCHEMA_VERSION
  const result = checkSchemaVersion(db, "/nonexistent/path/not/used.db");
  expect(result).toBe(true);
  db.close();
});
```

Note: The `dbPath` argument is not used when the function returns `true` (no deletion). Passing a nonexistent path is safe.

## Where to Insert
