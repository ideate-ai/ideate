# WI-164: Migration — interview files

## Context

`runMigration` does not migrate interview files. The gap-analyst identified this as MR2: the `.ideate/interviews/` subdirectory exists in `IDEATE_SUBDIRS` but nothing is ever written there. Interview files contain original planning conversations and refinement decisions that are architecturally significant.

WI-146 specification listed interview migration as step 11.

## What to migrate

From the source project root:
- `steering/interview.md` (if it exists) — the legacy monolithic interview file
- `steering/interviews/**/*.md` — per-cycle interview files in the refactored layout

Target location: `.ideate/interviews/` preserving the relative subdirectory structure.

Examples:
- `steering/interview.md` → `.ideate/interviews/interview.yaml`
- `steering/interviews/refine-018/_full.md` → `.ideate/interviews/refine-018/_full.yaml`
- `steering/interviews/refine-018/_general.md` → `.ideate/interviews/refine-018/_general.yaml`

## Function structure

```typescript
async function migrateInterviews(
  ctx: MigrationContext,
  sourceDir: string,
  ideateDir: string
): Promise<void> {
  // 1. Check for legacy steering/interview.md and migrate if present
  // 2. Glob steering/interviews/**/*.md and migrate each
}
```

## YAML output format

```yaml
id: {relative-path-slug}   # e.g., "interviews/refine-018/_full"
type: interview
cycle: {NNN if derivable from path, else null}
source_path: steering/interviews/refine-018/_full.md
content: |
  {raw markdown content}
content_hash: {sha256}
token_count: {approx}
```

For the legacy `steering/interview.md`, use `id: "interviews/legacy"` and `cycle: null`.

For `steering/interviews/{name}/*.md`, derive the cycle from the directory name if it matches `refine-{NNN}` pattern (e.g., `refine-018` → cycle 18).

## Handling missing files

If `steering/interview.md` does not exist and `steering/interviews/` does not exist, call returns without writing any files. Do not fail.

## Call site in runMigration

Add after `migrateSteeringArtifacts`:
```typescript
await migrateInterviews(ctx, sourceDir, ideateDir);
```
