# WI-207: Agent Definitions Update

## Changes

### agents/decomposer.md
Lines ~97-108: Rewrite acceptance criteria guidance.
Current: "Avoid criteria that require human judgment... this should be rare."
New: Per GP-13, identify validation method for each criterion:
- Objective criteria → machine verification (test, type check, structural assertion)
- Subjective criteria → human-in-the-loop (user approval, visual review, A/B comparison)
- Each criterion must state its validation method
- Neither method is exceptional — use whichever is appropriate

### agents/code-reviewer.md
Ensure the reviewer handles subjective criteria appropriately:
- When reviewing work items with human-validated criteria, the reviewer should note these as "requires human review" rather than marking them as gaps
- Machine-verifiable criteria: verify directly
- Human-verified criteria: confirm they are documented with an approval authority
