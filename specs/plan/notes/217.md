# WI-217: Executive Reporting Script

## Design

`scripts/report-executive.sh` — high-level summary for project stakeholders.

### Sections:

1. **Project Summary** — Aggregate stats: total work items completed, total cycles, total agent spawns, total wall-clock time. Single table.

2. **Quality Metrics** — Latest cycle: finding counts by severity, trend vs previous cycle. First-pass acceptance rate (latest cycle). Overall rework rate.

3. **Cost Summary** — Total tokens consumed (input + output). Estimated cost (if available). Average tokens per work item. Average tokens per cycle.

4. **ROI Indicators** — Derived metrics:
   - Rework rate trend: is rework_count per item increasing or decreasing?
   - Convergence speed trend: are brrr cycles needed decreasing?
   - Tokens-per-finding: total tokens / total findings (lower = more efficient)
   - First-pass rate trend: improving, stable, or degrading?

### Formatting
Compact output suitable for pasting into a status update. Use markdown tables throughout.
