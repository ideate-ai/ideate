# WI-157: Complete migration script тАФ journal, archive cycles, metrics

## Context

Spec-adherence S1 and gap-analysis G1 flagged three missing WI-146 conversion steps: journal.md, archive/cycles/{NNN}/*.md, and metrics.jsonl. D-62 decided that archive markdown is replaced by YAML finding objects in `.ideate/cycles/{NNN}/findings/`.

## Dependencies

- WI-155 (directory structure must match what this writes to)
- WI-156 (module-level state must be fixed before adding more code)

## Implementation

### migrateJournal(ctx)

Parse `journal.md` into individual entries. Each entry starts with `## [{phase}] {date} тАФ {title}`. For each entry:

```yaml
id: J-{cycle}-{seq}
type: journal_entry
phase: execute    # or review, refine, etc.
date: "2026-03-24"
cycle_created: {cycle}
title: "Work item 143: .ideate/ directory scaffolding"
content: |
  Status: complete with rework
  Rework: 2 minor findings fixed...
content_hash: {sha256 of canonical JSON}
token_count: {Math.floor(canonical.length / 4)}
```

Write each to `.ideate/cycles/{cycle}/journal/J-{cycle}-{seq}.yaml`. Create the cycle directory if it doesn't exist.

Cycle detection: extract from the phase tag context (e.g., `[execute]` entries belong to the most recent `[refine]` cycle). If cycle cannot be determined, use `000`.

### migrateArchiveCycles(ctx)

For each `archive/cycles/{NNN}/` directory, parse each markdown review file. Each finding (identified by `### C1:`, `### S1:`, `### M1:` headings) becomes a finding YAML object:

```yaml
id: F-{NNN}-{seq}
type: finding
cycle: {NNN as number}
reviewer: code-reviewer   # derived from filename (code-quality.md тЖТ code-reviewer)
severity: critical         # derived from heading prefix (C тЖТ critical, S тЖТ significant, M тЖТ minor)
title: "Watcher ignored pattern silences all .ideate/ events"
description: |
  {issue text}
file_path: "mcp/artifact-server/src/watcher.ts"
line: 24
suggestion: |
  {suggested fix text}
addressed_by: null
content_hash: {sha256}
token_count: {estimate}
```

Write to `.ideate/cycles/{NNN}/findings/F-{NNN}-{seq}.yaml`.

File-to-reviewer mapping:
- `code-quality.md` тЖТ `code-reviewer`
- `spec-adherence.md` тЖТ `spec-reviewer`
- `gap-analysis.md` тЖТ `gap-analyst`

### migrateMetrics(ctx)

Copy `metrics.jsonl` to `.ideate/metrics.jsonl` if it exists. No transformation needed.
